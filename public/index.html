Welcome to Julius-Anime-API

Your gateway to anime data

Anime API that brings your favorite anime to life!

Explore anime categories, top lists, schedule, and more with just a few clicks.

Available Routes:

  • /spotlight - Spotlight anime
  • /trending - Trending anime
  • /top-10 - Top 10 anime
  • /genres - List of genres
  • /genre/:genre - Get anime by genre
  • /estimated-schedule - Estimated anime schedule
  • /producer/:producer - Get anime by producer
  • /episode-servers - Episode servers (e.g., `/episode-servers?episodeId=dandadan-19319?ep=128656`)
  • /episode-srcs - Episode sources (e.g., `/episode-srcs?id=dandadan-19319&ep=128549&category=sub&server=vidstreaming`)
  • /hls-proxy - HLS proxy for Gogoanime (e.g., `/hls-proxy?url=https://www110.anzeat.pro/streamhls/156eaab7f95a98a9d12644f457a04f9c/ep.1.1728164109.m3u8`)
  • /quality-proxy - Default quality proxy for Gogoanime
  • /segment-proxy - Segment proxy for quality proxy
  • /hianime-hls-proxy - HLS proxy for HiAnime (e.g., `/hianime-hls-proxy?url=https://ea.netmagcdn.com:2228/hls-playback/b712013e859a1edd96e17ed28d761c17dbb82de2ce8f9e1e7f63f61c4bef63430715916f48931ef9e459e3c1cee6998d11d0e09babace509710d6c17c027dd9327358244439f9bdeaa456a5c8c2cfb31f35f22d21b325afaf5d213edf16e2254bdaa915b515761fa3375f956a4a07927dc2bc10cce8183a216864172f9d5f98d4ed207ee20384f40e8bd7c82d9f52d85/master.m3u8`)
  • /hianime-quality-proxy - Default quality proxy for HiAnime
  • /hianime-segment-proxy - Segment proxy for quality proxy for HiAnime
  • /thumbnails-proxy - Thumbnails URL proxy
  • /webvtt-proxy - WEBVTT proxy
  • /search - Search anime (e.g., `/search?keyword=one%20piece`)
  • /:category - Get anime by category
  • /az-list/:letter - AZ list by letter
  • /info/:infoId - Get information about anime (e.g., `/info/dandadan-19319`)
  • /related/:infoId - Get related anime
  • /recommended/:infoId - Get recommended anime
  • /episodes/:infoId - Get episodes for an anime
  • /seasons/:infoId - Get seasons for an anime
